A. Welcome to Luxury Apartment - Single property Template Documentation!

Thank you for purchase Luxury Apartment - Single property HTML5 Template!

I hope this template will be helpful for you - if you'll have any problems using it, please contact me via this support email. Feel free to send me your website URL, which has been created using this template - I'll be happy viewing it!

B. Template installation

This template has been written in HTML5, CSS3 and JavaScript (with jQuery). After downloading this template from your user panel on ThemeForest, you have to unzipped all files to some local folder on your computer.

All of the template files are placed in "luxury-apartment" directory. Select all template files and open your FTP Client (for example FileZilla). Then, make connection with your server (you may need your server name, FTP username and password).

Locate the domain folder, where you want to install this template and then open "public_html" (or similar) directory. After that, make an upload of all files to selected folder - that's all!

C. HTML structure

In "luxury-apartment" directory you can find 9 template files, configured as same as demo page (but without demo images, which are not included):

  • index.html
    this is home page of this template

  • about-agent.html
    this is page configured for real-estate agent "about" page

  • about-company.html
    this is page dedicated for company description, highlights and history,

  • property-details.html
    page dedicated for property details, description etc.

  • contact.html
    this is contact page with Google Map

  • schedule-a-visit.html
    this is page dedicated for action form, for example: arrange a viewing

  • blog.html,
    blog-single.html
    this are blog pages - for blog posts list and for single blog post

  • 404-page.html
    this is page for 404 error (page not found)

D. CSS structure

Stylesheet file is located in "_assets/_css" directory. For all .html files you need to include only one .css file - "style.css". This template has been written using CSS3 - if you haven't used it before, go to this w3schools.com website - it's very helpful introduction.

Open "style.css" file (this is main stylesheet for Luxury Apartment - Single property HTML5 template) to see the CSS structure.

E. JavaScript

Luxury Apartment - Single property HTML5 template needs only one javascript file, located into "_assets/_js/" directory - functions.js; This is main javascript file with general functions and settings.

You shouldn't make any configuration changes in "functions.js" file - any customizations, like Google Map marker position should be done in HTML file.
